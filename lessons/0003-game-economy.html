💰 游戏经济设计：游戏的生命线

不懂经济的策划，不敢做数值型游戏。经济崩了，游戏就死了。

🎯 为什么策划必须懂经济设计？

"MMORPG 的本质是一个可以用剑杀人的聊天室，裹着一层经济系统。"

— Raph Koster, 《游戏设计之道》作者

想想你平时玩的手游——能量、金币、钻石、体力、抽卡券、强化石、碎片……每一样都是经济设计。

某 MMO 开服半年后，金币价格崩盘。原因是：

  • 每日任务产出 10000 金币
  • 但每天的消耗只有 3000 金币（修装备+传送费）
  • 多余的金币无处可去
  • 半年后 10000 金币只能买一个面包
  • 新玩家："我打个怪掉的金币连修装备都不够？" → 流失
  • 游戏死亡。
核心洞察 经济系统是游戏的"血液系统"。如果血液不流动或通货膨胀，游戏就会"心脏病发作"。好的经济设计 = 可持续的资源循环 = 长寿的游戏。

1️⃣ 经济的三要素

所有游戏经济系统都由三个基本要素构成：

🏭 Source（产出）

资源从哪里来。

  • 副本掉落
  • 任务奖励
  • 每日产出
  • 付费购买
  • 玩家交易

🪣 Sink（消耗）

资源从哪里消失。

  • 强化/升级
  • 合成/锻造
  • 修理/维护
  • 手续费/税收
  • 装饰/外观

🔄 流通（流通）

资源如何在玩家之间流动。

  • 交易行
  • 赠送/邮寄
  • 掉落/拾取
  • 公会共享
🏭 产出 (Source) → 🔄 流通 (流通) → 🪣 消耗 (Sink)
如果 Source > Sink → 💸 通胀 如果 Source < Sink → 🥶 通缩 如果 Source ≈ Sink → ✅ 健康
牢记这条核心公式 每个策划都应该在脑子里刻着这条公式：Source − Sink = 经济健康度。 长期 Source − Sink > 0，必定通胀。 长期 Source − Sink < 0，必定通缩（玩家越来越穷，不想玩）。

2️⃣ 资源类型金字塔

大多数游戏的资源分 4 层，每一层的设计目的不同：

🥇 硬通货（Hard Currency）

充值货币 · 钻石/点券 · 来源唯一：付费
"这是游戏收入的直接来源，必须严格控制产出"

🥈 软通货（Soft Currency）

金币/铜钱 · 主要流通货币 · 日常产出
"连接玩家所有行为的通用资源"

🏅 代币（Token）

公会币 · 竞技场币 · 活动代币 · 专项消耗
"隔离经济，防止通兑——限制资源跨系统流动"

🪨 材料（Materials）

强化石 · 碎片 · 突破材料 · 特定用途
"控制成长节奏，限制交易范围"

🔑 关键设计原则：每层不互通

好的经济设计，不同层的资源不能自由兑换：

❌ 坏设计（通兑）

金币 ↔ 钻石 ↔ 材料

玩家发现直接用钻石买金币最划算 → 没人做日常 → 日常系统废了 → 内容消耗 <- 恶性循环

✅ 好设计（隔离）

金币 = 日常产出
钻石 = 付费获取
材料 = 副本掉落

每个系统有独立的产出闭环 → 玩家有理由参与所有系统

3️⃣ 通货膨胀：游戏经济的头号杀手

通胀不是"钱变多了"——是单位货币的购买力下降了。

📋 《魔兽世界》经济演变 (2004-2024)

版本金币价值原因
TBC (2007)1 金 ≈ 1 RMB产出有限，Sink 充足
WLK (2009)1 金 ≈ 0.3 RMB副本产出增加
MoP (2012)1 金 ≈ 0.05 RMB场景战役大量产金
Legion (2016)1 金 ≈ 0.003 RMB任务奖励膨胀 + 新 Sink 跟不上
Shadowlands (2020)1 金 ≈ 0.0005 RMB全面通胀

教训：每个版本都在增加 Source（新地图任务奖励、新副本掉落），但是 Sink 的增长没有跟上（修理费、传送费的涨幅远小于产出涨幅）。这是几乎所有 MMO 都遭遇的通胀死循环。

通胀的危害链

新玩家获得的金币买不起拍卖行的装备
↓
新玩家感到追赶无望
↓
新玩家流失
↓
新血不足 → 老玩家匹配不到队友
↓
老玩家也流失 → 服务器死亡
反直觉的事实 游戏通胀的最大受害者不是玩家——是新玩家。老玩家早已积累了大量财富，通胀对他们影响不大。但新玩家一进来看见物价飞涨，直接弃坑。通胀杀死的是游戏的新陈代谢能力。

4️⃣ Sink 的艺术：如何让资源"快乐地消失"

Sink 是游戏经济设计师最重要的工具。一个好的 Sink 是玩家愿意花钱、而且花了还开心的消耗途径。

Sink 类型例子玩家感受
成长消耗强化装备、升级技能😊 "我变强了"
装饰消耗时装、坐骑、家园😊 "我好帅"
社交消耗公会捐献、婚礼、送礼😊 "我和朋友一起"
维护消耗修理装备、传送费😐 "应该的"
手续费交易税、上架费😑 "可以接受"
错误成本洗点、重置天赋😤 "好贵……但不得不花"

Sink 设计的三个原则：

  1. 足够多——不能只有一个 Sink。经济崩溃往往是因为只有一个 Sink 被堵住了
  2. 有层次——有些 Sink 面向所有人（修理费），有些面向核心玩家（极限强化），有些面向土豪（外观）
  3. 伴随成长——最好的 Sink 是玩家"主动想花"而不是"被迫花"的
💡 经典案例：《COC 部落冲突》中，9 本 → 10 本升级费用比之前翻 3 倍。这是故意的——因为玩家到了 9 本，金币已经囤积了很多，需要一个大 Sink 来"抽干"他的库存。每次大版本更新，新的建筑等级就是一块巨大的 Sink。

5️⃣ 三种经典经济设计模式

🏗️ 模式 1：COC 模式 — Sink 绑定成长

打金 → 升级建筑（消耗金币） → 打更多金
资源仓库有上限 → 不打就浪费 → 被迫上线驱动

特点：每个成长节点就是一次 Sink。仓库上限是"强制循环"的设计工具。

🧩 模式 2：原神模式 — 资源分层隔离

资源用途可否通兑
原石（付费）抽卡→ 可换体力，但不能换经验书
摩拉（通用）强化、锻造、升级→ 几乎所有系统都需要
树脂（体力）副本准入→ 不可交易，天然上限
经验书角色升级→ 只能刷本获得
天赋书技能升级→ 特定日子开放（进一步限制）

特点：通过"资源隔离 + 体力限制"精确控制玩家的成长节奏。每种资源都有自己的 Source 和 Sink，互不干扰。

🤝 模式 3：MMO 模式 — 社交驱动流通

玩家 A 采药 → 挂拍卖行 → 玩家 B 购买 → 制作药水 → 卖回给 A
经济 = 社交的"骨架"

特点：玩家之间的交易本身创造"经济活力"。拍卖行/交易行是经济的"心脏"。

⚠️ 风险：当玩家发现"自给自足比交易更划算"，交易经济就死了。这就是为什么很多 MMO 限制玩家自产自用（如装备绑定、采集需要专业技能）。

6️⃣ 深度案例分析

📋 《COC》10 年经济的秘密

经济结构：

  • 金币：建造/升级建筑（主要 Sink）
  • 圣水：造兵/研究科技（另一个独立 Sink）
  • 暗黑重油：英雄/高级兵种（第三个 Sink）

聪明的设计点：

  1. 多 Sink 并行——金币、圣水、黑油各自有独立的消耗途径，不会相互通兑
  2. 仓库上限 = 强迫消耗——满了不打就被抢，所以你要么消耗掉，要么上线打（这就是留存）
  3. 进攻收益阶梯——低本打高本资源奖励高 → 鼓励越级挑战 → 保持挑战感
  4. 9 年后依然健康的秘密——每次大版本更新引入新的建筑等级（新 Sink），抽干老玩家的存量资源
💡 行业金句：COC 的设计师曾说——"我们的经济系统不是为了让玩家变富，而是为了让玩家永远觉得'还差一点'。"这种永远差一点的状态，就是游玩的驱动力。

步骤 2：找到 Source 和 Sink

Source 和 Sink 哪边更大？有通胀/通缩的迹象吗？你怎么判断的？

✅ 检验理解

在继续之前，请思考以下问题：

  1. 以你正在做或最熟悉的游戏为例，它的主要通货（金币/钻石/等）的 Source 和 Sink 分别是什么？你觉得目前经济是健康的还是通胀/通缩？为什么？
