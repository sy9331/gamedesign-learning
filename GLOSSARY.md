# 游戏策划 Glossary

## 基础概念

**核心循环 (Core Loop)** — 玩家在游戏中最基本的重复行为循环。例如 RPG 的"打怪→升级→打更强的怪"。核心循环定义了游戏的本质体验。

**系统思维 (Systems Thinking)** — 把游戏看作相互关联的系统集合，理解元素之间的因果关系和反馈回路，而不是孤立地看单个功能。

**反馈回路 (Feedback Loop)** — 系统中一个变量的变化会通过一系列链条影响自身。分为正反馈（放大变化）和负反馈（抑制变化）。

**正反馈循环 (Positive Feedback / Virtuous Cycle)** — 成功带来更多成功。例如：等级高→打更强的怪→获得更好的装备→变得更强→打更更强的怪。扩大玩家间的差距。

**负反馈循环 (Negative Feedback)** — 追赶机制，缩小差距。例如：落后方获得增益buff，领先方获得debuff。增加比赛的悬念。

**涌现 (Emergence)** — 简单的规则组合产生复杂且不可完全预测的行为。例如《塞尔达》中金属武器+雷电天气→引雷攻击。

**游戏经济 (Game Economy)** — 游戏中所有可交易、可获取、可消耗资源的系统总和。与现实经济类似，面临通胀、通缩、资源循环等挑战。

**内容规划 (Content Planning)** — 规划游戏上线后各阶段的新内容投放节奏，确保玩家始终有事可做且不会过快消耗。

**长线运营 (Long-term Operation)** — 以月/年为单位规划游戏的持续性运营策略，包括版本节奏、活动规划、用户留存策略等。

**商业化 (Monetization)** — 将游戏体验转化为收入的设计，包括付费点设计、商品定价、促销策略等。

**玩家心理学 (Player Psychology)** — 研究玩家在游戏中的动机、行为模式、决策偏好和情感反应的学科。

## 运营指标

**留存 (Retention)** — 玩家在首次体验后持续回归游戏的比例。通常用次日/7日/30日留存衡量。

**转化 (Conversion)** — 免费玩家转化为付费玩家的比例。

**生命周期价值 (LTV)** — 一个玩家在整个游戏生命周期内贡献的总收入。

**DAU/MAU** — Daily/Monthly Active Users，日活跃用户/月活跃用户。

**付费渗透率** — 付费用户占总活跃用户的比例。

**ARPU / ARPPU** — ARPU = 总收入/总用户数；ARPPU = 总收入/付费用户数。
