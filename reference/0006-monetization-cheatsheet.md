# 商业化思维速查表

## 付费金字塔

```
      💎 炫耀付费
    (外观/称号/限定)
   ──────────────────
     ⚔️ 强度付费
   (数值/属性/效率)
  ──────────────────
    🛠️ 便利付费
  (体力/加速/背包)
 ──────────────────
    🌟 体验付费
 (游戏本身/通行证)
```

## 核心公式

```
LTV = 平均付费金额 × 付费周期
ARPU = 总收入 ÷ 总用户数
ARPPU = 总收入 ÷ 付费用户数
付费渗透率 = 付费用户 ÷ 总用户
```

## 付费模式对比

| 模式 | 代表 | 优点 | 缺点 |
|:---|:---|:---|:---|
| **买断制** | 塞尔达、老头环 | 一次性收入，无后续压力 | 收入上限低 |
| **订阅制** | Xbox Game Pass | 稳定现金流 | 需要持续高质量内容 |
| **F2P + 内购** | 原神、COC | 低收入门槛高天花板 | 设计不当=剥削性 |
| **Battle Pass** | Fortnite | 持续活跃+收入 | 需要赛季制支撑 |
| **广告变现** | 三消、超休闲 | 免费玩家的价值 | 影响体验 |

## 付费点类型

| 类型 | 玩家感受 | 健康度 |
|:---|:---|:---|
| 卖外观 | "我好看" | ✅ 最健康 |
| 卖便利 | "省时间" | ✅ 良性 |
| 卖数值 | "我更强" | ⚠️ 需要谨慎 |
| 卖通关 | "不花钱过不去" | ❌ 剥削性 |
| 卖随机 | "赌一把" | ⚠️ 伦理风险 |

## 免费玩家→付费玩家路径

```
体验乐趣 → 遇到瓶颈 → 感受限制
              ↓
        第一次付费（低门槛）
              ↓
        获得正向体验提升
              ↓
        形成付费习惯
              ↓
        付费身份认同（"我是月卡党"）
```

## 商业化健康度检查清单

- [ ] 免费玩家能否体验到完整的核心循环？
- [ ] 付费玩家是否获得了正面的体验提升？
- [ ] 是否存在"不付钱就玩不下去"的关卡？
- [ ] 付费差距是否在可接受范围内？
- [ ] LTV > CAC（获客成本）
- [ ] 付费渗透率在健康范围（通常 5-15%）
- [ ] 是否有除了内购以外的收入来源？

## 定价心理学

| 策略 | 原理 | 例子 |
|:---|:---|:---|
| **锚定** | 高价锚点让中价看起来便宜 | 648 → 328 |
| **诱饵** | 一个不划算的选择让你选另一个 | 30元/98元/128元(推荐) |
| **损失框架** | "省了"比"花了"更好听 | "省70元" vs "付30元" |
| **分期感知** | 把大额拆小额 | "每天只需 2 块钱" |
| **社会证明** | 别人买了所以值得买 | "已售 10000+ 份" |

## 常见商业化指标

| 指标 | 健康范围 | 含义 |
|:---|:---|:---|
| 付费渗透率 | 5-15% | 付费玩家占比 |
| 月卡占比 | >30% | 稳定付费用户基础 |
| 首日付费率 | >2% | 品质和付费引诱的验证 |
| 平均付费频次 | >2次/月 | 付费习惯形成 |
| 付费LTV | >获客成本×3 | 商业可持续 |
