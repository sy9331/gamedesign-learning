# 决策能力速查表

## 决策层次金字塔

```
      🏛️ 战略决策
   (3-12 月 · 不可逆)
  ───────────────────
     ⚔️ 战术决策
   (1 周-3 月 · 可调整)
  ───────────────────
     ⚡ 执行决策
   (每天 · 可快速迭代)
```

## OODA Loop 决策框架

```
观察 (Observe) → 判断 (Orient) → 决策 (Decide) → 行动 (Act)
                                         ↑            │
                                         └────────────┘
```

| 阶段 | 在游戏策划中的含义 |
|:---|:---|
| **Observe** | 收集数据（DAU、留存、付费、玩家反馈、竞品动态） |
| **Orient** | 分析数据，识别模式，理解上下文（为什么涨/为什么跌） |
| **Decide** | 在当前情境下选择最优方案（多个方案中选一个） |
| **Act** | 执行并快速验证 |

## 决策检查清单

### 决策前
- [ ] 这个问题有"标准答案"吗？如果有 → 不需要"决策能力"，去查资料即可
- [ ] 这个问题有多重要？（决定投入多少认知资源）
- [ ] 决策的代价是什么？改了能改回来吗？
- [ ] 我需要在什么时候之前做决定？（deadline）

### 决策中
- [ ] 我有什么信息？还缺什么信息？
- [ ] 我有多少个选项？（少于 2 个 = 不是决策）
- [ ] 每个选项的最佳结果和最差结果是什么？
- [ ] 我最希望哪个结果发生？(揭露你的偏好)
- [ ] 反着选会怎样？(反向思考)

### 决策后
- [ ] 我如何验证决策是对的？
- [ ] 我的止损点在哪里？（什么情况下必须回头）
- [ ] 这次决策我学到了什么？

## 常见决策陷阱

| 陷阱 | 描述 | 如何避免 |
|:---|:---|:---|
| **确认偏误** | 只看支持自己想法的证据 | 主动寻找反面证据 |
| **沉没成本** | 因为已经投入了所以继续 | 问"如果现在从零开始，我还会这么做吗？" |
| **锚定效应** | 被第一个信息影响 | 先独立思考，再看别人的建议 |
| **群体思维** | 为了和谐而妥协 | 指定一个人扮演"反对者" |
| **可得性启发** | 因为最近见过就认为概率高 | 用数据而不是直觉判断概率 |
| **过度自信** | 高估自己的判断准确率 | 记录预测并回顾准确率 |

## 决策复盘模板

```
1. 这个决策的目标是什么？
2. 实际结果是什么？（用数据说话）
3. 预期和实际之间差了什么？
4. 我当时忽略了什么信息？
5. 如果让我重新做一次，我会怎么做？
6. 这次决策教会了我什么？（写入学习记录）
```

## 信息完整度决策矩阵

```
                   信息完整度高
                       │
         ⚠️ 小心决策     │    🎯 可按标准流程
         (Google 式)    │    (麦当劳式)
                       │
    ────────────────┼─────────→ 决策重要性
         ❌ 先收集信息   │    ✅ 大胆决策
         (不要乱猜)     │    (创业式)
                       │
                   信息完整度低
```

## 一句话总结

> 决策不是"选对的"——决策是"在你不知道什么是对的的时候，仍然敢选一个方向，并为自己的选择负责。"
