# 玩家心理学速查表

## 玩家动机四大模型

### 1. Bartle's Taxonomy (巴图分类法)
```
社交者  ─┬─  探索者
         │
成就者  ─┴─  杀手
```
- **成就者**：追求等级、装备、排名、成就
- **探索者**：追求发现新内容、新玩法、彩蛋
- **社交者**：追求与其他玩家互动、公会、聊天
- **杀手**：追求竞争、PVP、打败其他人

### 2. Self-Determination Theory (自我决定论)
- **自主性 (Autonomy)**：我能自己选择怎么玩
- **胜任感 (Competence)**：我越玩越擅长
- **归属感 (Relatedness)**：我属于这个社区

### 3. Yee's Motivations (Nick Yee 动机模型)
- **成就**：进步、机制优化、竞争
- **社交**：社交化、关系、团队协作
- **沉浸**：发现、角色扮演、自定义、逃避现实

### 4. 四驱动机型 (Quantic Foundry 模型)
```
行动导向  ←→  策略导向
         ↓
   沉浸/叙事  ←→  社交/竞争
```

## 游戏设计中的心理学偏差

| 偏差 | 原理 | 游戏应用 |
|------|------|---------|
| **损失厌恶** | 失去的痛苦 > 得到的快乐 | 每日签到（断签损失）、限时活动 |
| **沉没成本** | 投入越多越难放弃 | 等级系统、长期任务进度 |
| **随机奖励** | 不确定的奖励更上瘾 | 抽卡、开箱、掉落 |
| **蔡格尼克效应** | 未完成的事更让人惦记 | 进度条、未完任务标记 |
| **禀赋效应** | 拥有的东西价值更高 | 自选初始角色、自定义 |
| **锚定效应** | 第一印象锚定价格 | 原价→折扣、礼包对比 |
| **社会证明** | 别人在做=值得做 | 排行榜、好友动态、世界聊天 |

## 心流通道 (Flow Channel)

```
         焦虑区
          ↑
   高     │  ░░░░
  挑战    │░░░░░░░░  ← 心流通道
          │████████→
          ↓   技能 →
         低         高
         厌倦区
```
- 挑战 > 技能 → 焦虑
- 技能 > 挑战 → 无聊
- 挑战 ≈ 技能 → 心流

## 内在动机 vs 外在动机

| 内在动机 | 外在动机 |
|---------|---------|
| 因为好玩而玩 | 为了奖励而玩 |
| 持续性强 | 奖励消失即停止 |
| 产生心流 | 产生依赖 |
| MMO/RPG 为主 | 三消/挂机为主 |

**⚠️ 过度使用外在动机会扼杀内在动机**（过度理由效应）

## 免费玩家→付费玩家心理路径

```
体验乐趣 → 投入时间 → 沉没成本积累
    ↓                        ↓
想获得更多 ← 感受限制 ← 遇到瓶颈
    ↓
第一次付费（冲动/便利/社交压力）
    ↓
付费习惯形成 → 付费身份认同
```

## 分析框架

分析任何游戏的心理学设计：
1. **目标玩家是谁？**（Bartle 类型）
2. **核心动机是什么？**（SDT 三要素）
3. **什么让人上瘾？**（哪个认知偏差被利用了）
4. **付费点设计？**（卖便利/卖强度/卖身份？）
5. **心流管理？**（难度曲线是否平滑？）
